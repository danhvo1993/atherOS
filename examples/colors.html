Colors

The color system is based on Microsoft's Fluent 2 Design System, providing a comprehensive palette for creating accessible and visually consistent interfaces.

Usage

.my-element {
    color: var(--color-blue);
    background-color: var(--color-grey-02);
}

Accessibility

All color combinations in the system meet WCAG 2.1 AA standards for contrast ratio. When using colors, ensure:

  • Text colors have sufficient contrast with their background (4.5:1 for normal text, 3:1 for large text)
  • Don't rely solely on color to convey information
  • Provide alternative visual cues (icons, patterns) alongside color

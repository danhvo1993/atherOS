Spacing

A consistent spacing scale helps create visual rhythm and hierarchy in interfaces. AtherOS uses a modular spacing scale based on 4px units.

Usage

.my-element {
    margin: var(--spacing-md);
    padding: var(--spacing-sm) var(--spacing-lg);
}

Spacing Scale

Extra Small (XS)
var(--spacing-xs)
Used for tight spaces between icons and text
Small (SM)
var(--spacing-sm)
Common padding for buttons and form elements
Medium (MD)
var(--spacing-md)
Standard spacing between components
Large (LG)
var(--spacing-lg)
Section padding and larger gaps
Extra Large (XL)
var(--spacing-xl)
Major section spacing
2X Large (2XL)
var(--spacing-2xl)
Page-level spacing and margins

Best Practices

  • Use consistent spacing within component types
  • Maintain proper hierarchy with spacing scale
  • Consider content density when choosing spacing values
  • Use larger spacing for separation of major sections
  • Maintain breathing room around interactive elements

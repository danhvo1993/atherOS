Buttons

Buttons are interactive elements that allow users to trigger actions. The Fluent Design System provides various button styles for different contexts and hierarchies.

Usage

<button class="button button--primary">
    Primary Button
</button>

Button Variants

Primary
Main call-to-action, one per major view
button button--primary
Secondary
Alternative actions, supporting tasks
button button--secondary
Accent
Highlighted actions, drawing attention
button button--accent
Danger

button button--danger
Ghost
Subtle actions, minimal visual weight
button button--ghost

Accessibility

Buttons are designed with accessibility in mind:

  • All buttons are keyboard focusable
  • High contrast focus indicators
  • ARIA labels for icon-only buttons
  • Proper color contrast ratios
  • Loading states are announced to screen readers

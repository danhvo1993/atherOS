Typography

The typography system is based on Microsoft's Fluent 2 Design System, providing a clear hierarchy and consistent reading experience across all platforms.

Usage

.my-element {
    font-family: var(--font-family-base);
    font-size: var(--font-size-base);
    font-weight: var(--font-weight-regular);
    line-height: var(--line-height-base);
}

Accessibility

The typography system is designed with accessibility in mind:

  • All text meets WCAG 2.1 AA standards for contrast ratio
  • Base font size is responsive and adjusts based on user preferences
  • Line heights ensure proper readability
  • Font weights maintain legibility across different screen types
